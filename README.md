# Seoul-airbnb-Analysis-Project
## 서울 Airbnb Opportunity Map: 입지·상품·운영 레버 기반 RevPAR극대화 가이드


프로젝트 요약: 서울 Airbnb 인프라 및 성과 지표 분석과 교통-관광지-유동-경쟁도를 결합해 유망 입지와 운영 전략을 도출


## 목표
1. 교통 접근성.유동(지하철).장소 프리미엄(120장소).경쟁도.숙소 스펙을 결합한 Opporyunity Score 상위 지역을 지도 기반 대시보드로 시각화하고 TOP 후보지를 추천한다.
2. 최근 90일 기준으로 계산한 RevPAR로 예측/설명 모델을 구축해 Spatial CV 기반 MAE/RMSE/R²로 정량 검증한다.
3. 입지 유형별 추천 숙소 타입과 운영 레버(즉시예약/최소박수/신뢰지표)의 기대효과를 플레이북/시뮬레이션 형태로 제시한다.


## 문제정의
1. 성공적인 에어비앤비 숙소 운영을 위해 잠재적 고객인 관광 고객의 특징은 지역별로 어떤 차이가 있을까?
2. 성공적인 에어비앤비 숙소 창업 및 운영을 위해서 가장 최적의 숙소 운영 전략은 무엇인가?


## 활용 데이터
1. 서울 Airbnb masked DATA:

화장실 수, 침실 수, 침대 수 등 숙소 내부 인프라 지표

청소비, 추가 인원 비용, 최대 수용 인원, 즉시 예약 가능 여부, 최소 숙박 일수, 슈퍼호스트 여부 등 숙소 운영 관련 지표

리뷰 수, 사진 수, 종합 평점 등 고객 신뢰도 관련 지표

자치구, 숙소 위치 좌표 등 숙소 위치 관련 지표

점유율, 매출액, 객실 당 매출 등 수익 관련 지표

2. 서울시 공공데이터:

숙박업 관련 인허가 정보 데이터

자치구별 내/외국인 방문자 수 및 숙박/체류시간 데이터

서울시 관광지 관련 데이터

유동인구 관련 데이터

교통 및 문화행사 관련 데이터

글로벌 키워드 관련 데이터

글로벌 SNS 관련 데이터

공공데이터로 서울 내 관광 특성 & 방문객 특성 확인, 서울 airbnb 데이터로 현재 운영 중인 숙소 컨디션 확인, 두 데이터 교차 분석으로 추천 입지 구체화  & 지역별 마케팅 전략 도출


## 기대효과
공공데이터로 서울 내 관광 특성 & 방문객 특성 확인, 서울 airbnb 데이터로 현재 운영 중인 숙소 컨디션 확인, 두 데이터 교차 분석으로 추천 입지 구체화  & 지역별 마케팅 전략 도출

1. 서울 지역별 관광 고객의 특성 파악을 통해 에어비앤비의 잠재 고객 특징을 파악할 수 있음.
2. 성과가 높은 에어비앤비 숙소 분석을 통해 운영 전략 구축 가능
3. 지역별 에어비앤비 숙소 전략을 통해 성공적인 초기 온보딩 or 운영 전략 변경 가능

## 데이터 전처리 과정
1. 서울 Airbnb masked DATA:

욕실 수 15개 초과, 침실 수 20개 초과, 최소숙박일수 1년 초과, 리뷰개수 1000개 초과, 추가인원비용 200,000원 초과, 각 컬럼별로 고유값이 1개인 행 제거

매출 0인 데이터 → 비중 30% 차지, 수익 관련 레버 왜곡 우려 → 삭제

매출, 객실 요금과 같이 연속 데이터는 로그 스케일 변환

범주형 데이터는 윈저화

제거 전 행 개수: 32101 개 → 제거 후 행 개수: 20000 개

2. 서울시 공공데이터:

좌표 기준값 변경(epsg5174 → epsg5179) & epsg4326 좌표 추가

좌표명 변경(epsg5179 → GRS80TM, epsg4326 → WGS84)

자치구·행정동 컬럼이 존재할 때, 좌표값 데이터와 매핑하여 결측치 보완(반대의 경우도 동일하게 진행), 진행 후에도 보완되지 않았을 경우에는 제거

'폐업', '직권말소', '등록취소' 등 사용하지 못하거나 관련이 적은 컬럼 제거

체류시간 분 → 시간 변환

연/월 분리 및 분기 컬럼 생성

장소명, 좌표값 등의 중복 데이터 제거

유사 형식 or 연속적인 데이터들 통합
